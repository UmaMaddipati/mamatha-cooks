# 🍲 Mamatha Cooks

Welcome to **Mamatha Cooks**! This is a modern, responsive e-commerce web application built for a seamless food ordering experience. 

## 🚀 Features
- **User Authentication**: Secure Sign Up and Login workflows.
- **Product Catalog**: Browse through a variety of delicious products.
- **Shopping Cart**: Add items to your cart, select varying product weights (e.g., 1/4kg, 1kg), and manage quantities.
- **Checkout Process**: Smooth checkout modal and order finalization.
- **Order Tracking**: Keep track of user order history and fulfillment.
- **Logistics Integration**: Integrated with Shiprocket for dynamic weight calculations and delivery status.

## 🛠️ Technology Stack
- **Frontend Framework**: React.js with Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: Shadcn UI / Radix UI
- **Routing**: React Router
- **Animations**: Framer Motion

## 📦 Getting Started

### Prerequisites
Make sure you have Node.js and npm (or bun) installed on your system.

### Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/UmaMaddipati/mamatha-cooks.git
   cd mamatha-cooks
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or
   bun install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open in browser:**
   Navigate to `http://localhost:8080/` (or whichever port Vite starts on) to view the application!
